Public Endpoints

Endpoint Method Accessible By JSON Description
/test GET Public N/A The /test endpoint (Postman: checkIfApiOnline (Public)) shows if the FOUNDFAVE API is online.
/info GET Public N/A The /info endpoint (Postman: showApiInfo (Public)) gives you a list of all available endpoints.
/queries GET Public N/A The /queries endpoint (Postman: showAllAvailableQueries (Public)) shows a list of all available queries: Characters and movies.
/documentation/index.html GET Public N/A The /documentation/index.html endpoint (Postman: API documentation (Public)) shows a list of all available endpoints in detail. E.g. This page.

Characters Endpoints

Endpoint Method Accessible By JSON Description
/characters GET Admin, User N/A The /characters endpoint (Postman: getAllCharacters) allows you to view all Marvel characters currently in de database.
/characters/{characterId} GET Admin, User N/A The /characters/{characterId} endpoint (Postman: getCharacterById) allows you to view a specific Marvel character with an id.
/characters/search/starting-with GET Admin, User N/A The /characters/search/starting-with endpoint (Postman: findCharactersByNameStartingWith) allows you to search for characters with a query parameter. This method searches for characters starting their name as filled in as query parameter.
/characters/search/contains GET Admin, User N/A The /characters/search/contains endpoint (Postman: findCharactersByNameContains) allows you to search for characters with a query parameter. This method searches for characters whose name contains that which was filled in as query parameter.
/characters/search/sorted-asc GET Admin, User N/A The /characters/search/sorted-asc endpoint (Postman: findCharactersByNameSortedAsc) allows you to search for characters with a query parameter. This method returns the search results in ascending order.
/characters/search/sorted-desc GET Admin, User N/A The /characters/search/sorted-desc endpoint (Postman: findCharactersByNameSortedAsc) allows you to search for characters with a query parameter. This method returns the search results in descending order.
/characters/search/actor-name GET Admin, User N/A The /characters/search/actor-name endpoint (Postman: findCharactersByActorNameContains) allows you to search for characters with a query parameter. This method searches for characters that are played by certain actors and whose name contains that which was filled in as query parameter.
/characters/character/{characterId} PUT Admin {
"characterAliasName": "Hulk",
"characterRealName": "Robert Bruce Bannerz",
"characterActorName": "Mark Ruffalo",
"characterTitle": "The incredible Hulk",
"characterGender": "Male",
"characterSummary": "Exposed to heavy doses of gamma radiation, scientist Bruce Banner transforms into the mean, green rage machine called the Hulkz.",
"characterDescription": "Dr. Bruce Banner lives a life caught between the soft-spoken scientist he’s always been and the uncontrollable green monster powered by his rage."
} 		The /characters/character/{characterId} endpoint (Postman: updateCharacterById) allows you to update a character with a specific id.
/characters/associate/movie/{characterId} PUT Admin {
"id": 1000
} 		The /characters/associate/movie/{characterId} endpoint (Postman: associateMovieAndCharacter) allows you to combine a character and a movie. The character will then be added to that movie and that movie will be added to that character. The id in the JSON body is the id of the movie.
/characters POST Admin {
"characterAliasName": "She-Hulk",
"characterRealName": "Jennifer Walters",
"characterActorName": "Tatiana Maslany",
"characterTitle": "Savage She-Hulk",
"characterGender": "Female",

"characterDescription": " After being shot by a mobster and seriously injured, she was saved by a blood transfusion from Bruce, and his gamma-irradiated blood mutated Jennifer into the She-Hulk, kicking off her adventuring career."
} 		The /characters endpoint (Postman: createCharacter) allows you to create a character with the specified values and attributes.
/characters/{characterId} DELETE Admin N/A The /characters/{characterId} endpoint (Postman: deleteCharacterById) allows you to delete a character with a certain id. Note: Currently, characters assigned to users (Favorites) or movies, are not allowed to be deleted. Use the removeFavoriteCharacterFromUser and disassociateMovieAndCharacter methods before trying to remove the character.
/characters/disassociate/movie/{characterId} DELETE Admin {
"id": 1005
} 		The /characters/disassociate/movie/{characterId} endpoint (Postman: disassociateMovieAndCharacter) allows you to disassociate a character and a movie. The character will then be removed from that movie and that movie will be removed from that character. The id in the JSON body is the id of the movie.

Contact Forms Endpoints

Endpoint Method Accessible By JSON Description
/contactforms GET Admin N/A The /contactforms endpoint (Postman: getAllContactForms) allows you to see a list of all submitted contact forms.
/contactforms/search/name GET Admin, User N/A The /contactforms/search/name endpoint (Postman: findContactFormsByNameContains) allows you to find a contact form by entering a name as a query parameter or part of a name.
/contactforms/search/email GET Admin, User N/A The /contactforms/search/email endpoint (Postman: findContactFormsByEmailContains) allows you to find a contact form by entering an e-mail address or part of an e-mail address as a query parameter.
/contactforms/search/comments GET Admin, User N/A The /contactforms/search/comments endpoint (Postman: findContactFormsByCommentsContains) allows you to find a contact form by entering a keyword, or part of a keyword, as a query parameter, that may exist in a comment field within a contact form.
/contactforms/{contactFormId} GET Admin, User N/A The /contactforms/{contactFormId} endpoint (Postman: getContactFormById) allows you to find a contact form by entering its id.
/contactforms/post POST Admin, User {
"name": "Novi",
"email": "test@novi.nl",
"comments": "Dit is commentaar"
} 		The /contactforms/post endpoint (Postman: createContactForm) allows you to create and post a contact form using the predefined JSON body.
/contactforms DELETE Admin N/A The /contactforms endpoint (Postman: deleteAllContactForms) allows you to delete all submitted contact forms. Admin only as this is a very powerful feature!
/contactforms/{contactFormId} DELETE Admin, User N/A The /contactforms/{contactFormId} endpoint (Postman: deleteContactFormById) allows you to delete a contact form with a certain id.

File Endpoints

Endpoint Method Accessible By JSON Description
/download-profile-image/{fileName} GET Public N/A The /download-profile-image/{fileName} endpoint (Postman: downloadProfileImage (Public)) allows you to download a desired file, a profile image.
/download-character-image/{fileName} GET Public N/A The /download-character-image/{fileName} endpoint (Postman: downloadCharacterImage (Public)) allows you to download a desired file, a character image.
/download-movie-image/{fileName} GET Public N/A The /download-movie-image/{fileName} endpoint (Postman: downloadMovieImage (Public)) allows you to download a desired file, a movie image.
/upload-profile-image/{profileId} POST Admin, User N/A The /upload-profile-image/{profileId} endpoint (Postman: uploadProfileImageById) allows you to upload a desired image to a certain profile with an id. For best results upload a file with the .jpg or .png file format.
/upload-character-image/{characterId} POST Admin N/A The /upload-character-image/{characterId} endpoint (Postman: uploadCharacterImageById) allows you to upload a desired image to a certain character with an id. For best results upload a file with the .jpg or .png file format.
/upload-movie-image/{movieId} POST Admin N/A The /upload-movie-image/{movieId} endpoint (Postman: uploadMovieImageById) allows you to upload a desired image to a certain movie with an id. For best results upload a file with the .jpg or .png file format.
/delete-profile-image/{profileId} DELETE Admin, User N/A The /delete-profile-image/{profileId} endpoint (Postman: deleteProfileImageById) allows you to delete a profile with a certain id.
/delete-character-image/{characterId} DELETE Admin N/A The /delete-character-image/{characterId} endpoint (Postman: deleteCharacterImageById) allows you to delete a character with a certain id.
/delete-movie-image/{movieId} DELETE Admin N/A The /delete-movie-image/{movieId} endpoint (Postman: deleteMovieImageById) allows you to delete a movie with a certain id.
